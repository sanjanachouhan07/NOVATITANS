# **App Name**: Crypto Things

## Core Features:

- Secure User Authentication: Allow users to create accounts and securely log in, storing their unique progress and game state.
- Interactive Introduction Level: Guides players through the game's dark narrative, teaches basic cryptographic concepts, and introduces hint system mechanics through practice problems.
- Dynamic Cryptography Problem Generator: An AI tool that intelligently generates unique cryptographic challenges for each level by applying specified algorithms to varied inputs, ensuring fresh puzzles every session.
- AI-Powered Contextual Hint System: An AI tool that provides progressive, context-aware hints for each problem, including algorithm explanations, step-by-step guidance, and relevant tool suggestions.
- Multi-Level Game Progression: Seven main game levels, each focused on a specific cryptographic algorithm or mixed ciphers, with progression tracking and 'level complete' unlock mechanics.
- In-Game Decoder Tool Panel: An integrated utility allowing users to manually input encrypted text, select a cryptographic algorithm, and test potential decoding solutions.
- Progress and Reward System: Tracks user progress, completed problems, and awards in-game coins upon level completion, persistently stored in Firebase Firestore.

## Style Guidelines:

- Background color: Deep charcoal with a subtle reddish-brown tint (#2C2720), establishing a dark, eerie atmosphere.
- Primary color: Desaturated, dark rustic orange-brown (#5A4A3D), used for UI containers and less prominent text, creating an aged, mysterious feel.
- Accent color: Vibrant, luminous red (#E63333), creating a 'red glow' for interactive elements, highlights, warnings, and calls to action, drawing visual attention with intensity.
- All text: 'Space Grotesk' (sans-serif) for a modern, techy, slightly unsettling digital aesthetic, fitting the cryptographic theme.
- Utilize cryptic, minimalist icons and glyphs with digital interference effects to reinforce the dark, supernatural data mystery.
- Full-screen, immersive layouts with cinematic transitions between levels and sections, enhancing the narrative's tension. Subtle fog overlays and glitch effects will integrate into problem displays.
- Implement digital glitch effects on textual elements, subtle pulsating glows for important notifications, and fluid cinematic transitions between game states. Incorporate atmospheric sound cues for key events.
