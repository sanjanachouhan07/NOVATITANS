# Project Idea Report

## Project Title

CipherThings: AI-Powered Cryptography Adventure Game

## 1. Introduction

CipherThings is a gamified cybersecurity learning platform designed to teach cryptography through an immersive mystery-based adventure game. The system combines puzzles, storytelling, artificial intelligence, and interactive gameplay to help users understand encryption techniques while solving challenges in a dark suspense-themed environment.

The platform transforms traditional cybersecurity learning into an engaging game where users decode encrypted transmissions, solve cryptographic puzzles, defeat digital enemies, and progress through increasingly difficult levels.

## 2. Project Objective

* Teach cryptography concepts through gameplay
* Make cybersecurity learning interactive
* Provide hands-on experience with encryption algorithms
* Build problem solving and analytical skills
* Create a competitive learning environment with rankings and rewards

## 3. Target Users

* Cybersecurity students
* Computer science students
* Ethical hacking learners
* Puzzle and mystery game lovers

## 4. Theme Concept

The game is inspired by a dark mystery universe where encrypted signals are coming from a parallel digital dimension called "Cipher Void".

Players act as codebreakers who must decode transmissions and stop malicious cyber entities.

Theme characteristics:

* Dark neon red environment
* Suspense background button  sound/ voice integration on buttons
* Glitch animations
* Digital monsters created from corrupted code
* Retro terminal interface

## 5. Storyline

A mysterious cyber dimension called Cipher Void has opened inside the global network. Unknown digital entities are sending encrypted transmissions that could collapse communication systems worldwide.

Players join a secret cyber defense unit and must decode signals, unlock portals, and defeat cyber creatures by solving cryptographic puzzles.

Each solved challenge weakens the digital threat and unlocks new areas of the encrypted world.

## 6. Gameplay Flow

1. User creates account
2. User enters the game dashboard
3. Player selects a mission level
4. System generates cryptographic puzzle
5. Player analyzes encryption type
6. Player submits decrypted message
7. System validates answer
8. Player receives score, XP and rewards
9. New level unlocks

Boss levels appear after several missions.

## 20. Adaptive Music and Sound System

Music intensity will change automatically according to gameplay events to increase suspense and immersion.

### Music States

soft cyber suspense music

faster heartbeat‑style rhythm

Wrong Answer / System Error

Correct Answer
unlock sound with rising synth

Monster Encounter

Victory

### Audio Automation Logic

The system detects game events and switches music tracks dynamically.

Example logic:

if puzzle_started → play suspense track
if timer < 20 seconds → increase tempo
if boss_level → activate intense soundtrack
if puzzle_solved → play victory sound

### Sound Implementation Tools

Tone.js
Howler.js

These allow dynamic volume control, fading between tracks, and real‑time sound effects.
